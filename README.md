# Tienda de Barrio
Este es un proyecto de prueba 
Autor: Alan Araujo Jacome 
## Descripción 
Este proyecto es una aplicación diseñada para la gestión de una tienda de barrio, 
permitiendo controlar inventarios, registrar ventas y manejar clientes de forma sencilla.
Su objetivo es facilitar la administración diaria y mejorar la eficiencia del negocio.
## Pasos de Instalación y Ejecución
Como primer paso, entrar al link con este link podrias ingresar a git y desde ahí podrias descargarlo 
  1. git clone https://github.com/araujojacomealan/EJ-RG-2.git
Como segundo paso despues de instalarlo o descargarlo lo descomprimes el archivo que esta en esa carpeta
   2. EJ-RG-2
Y como tercer y ultimo paso abres la carpeta en VSC o desde VSC le das el comando ctrl+k+o para abrirlo y lo buscas 
    3.  Visual estudio code
## Caracteristicas de mi proyecto 
  * Registro y control de productos eh inventarios
  * Gestión de ventas y reportes diarios
  * Interfaz sencilla y agradable
